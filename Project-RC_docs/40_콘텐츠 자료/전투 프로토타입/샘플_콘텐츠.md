# 전투 샘플 콘텐츠 명세서

> 문서 상태: 초안  
> 기준 전투 모델: 플레이어 3인 파티 vs 보스 1마리  
> 목적: 프로토타입 구현에 바로 넣을 수 있는 최소 샘플 데이터를 제공한다

---

# 1. 개요

본 문서는 전투 프로토타입을 구현할 때 사용할 임시 캐릭터, 행동, 보스, 약점 데이터를 정의한다.  
내용은 최종 설정이 아니라 테스트용 샘플이다.  
기획팀이 세부 내용을 채우기 전까지 구현자가 기능 검증에 사용할 수 있는 기준값으로 둔다.

# 6. 샘플 전투 흐름

1. 전투 시작
2. 플레이어 3명은 모두 대기 상태
3. 보스 약점은 `?`, `?`로 표시
4. 플레이어가 화염구를 사용
5. 화염이 약점이면 `화염 1/3`으로 공개
6. 플레이어가 화염 속성을 2회 더 적중
7. `화염 3/3`이 되어 보스 기절
8. 기절 5초 동안 보스 행동 정지
9. 기절 종료 후 화염 약점 비활성화
10. 남은 약점 또는 새 약점 공략
11. 보스 HP 0 도달 시 승리

---

# 7. 테스트 포인트

이 샘플 콘텐츠로 아래를 확인한다.

- 3명 캐릭터를 관리하는 것이 과하게 바쁘지 않은가
- 1.5~4초 행동 시간이 선택 차이로 느껴지는가
- 약점이 공개되는 순간을 플레이어가 알아차리는가
- 3회 공략 후 기절이 충분한 보상으로 느껴지는가
- 보스 행동 압박이 약점 공략을 방해하되 불합리하지 않은가
