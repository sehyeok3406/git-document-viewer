## 1.1 캐릭터 설계 의도

마리아는 Un-Neet의 첫 번째 플레이 캐릭터다.

따라서 플레이어가 아래 행동을 자연스럽게 배우도록 설계한다.

1. 카드를 아끼지 않고 사용한다.
2. 저비용 카드와 0비용 카드를 연달아 사용한다.
3. 탄환을 소비해 공격한다.
4. 탄창을 비우면 재장전 보너스를 얻는다.
5. 시그니처 게이지가 완성되면 강한 연사 기술로 폭발한다.

마리아는 정교한 조준 계산보다 **“탄환이 있으면 쏘고, 비면 갈고, 다시 쏜다”**는 직관을 우선한다.

# 3. 고유 자원: 탄환(Bullet)

## 3.1 정의

탄환은 마리아 전용 자원이다.

- 현재 탄환 범위: `0 ~ 6`
- 전투 시작 시 탄환: `6`
- 일반 공격 카드 대부분은 탄환을 1발 소비한다.
- 탄환이 부족하면 해당 카드의 공격 효과는 사용할 수 없다.
- 일부 카드는 탄환을 소비하지 않는다.
- 재장전 카드 또는 탄창 소진 보상으로 탄환을 회복한다.

## 3.2 UI 표시

- 화면 상단 플레이어 정보 영역에 탄환 아이콘 6개를 표시한다.
- 채워진 탄환: 밝은 탄환 아이콘
- 소모된 탄환: 어두운 빈 탄환 아이콘
- 탄환 0 상태: UI에 `재장전 필요` 텍스트 또는 경고 효과를 표시한다.
- 탄환을 모두 소비한 순간에는 탄창 비움 보상 연출을 재생한다.

예시:

```text
탄환: ● ● ● ● ○ ○   (4 / 6)
```

## 3.3 공통 처리 규칙

### 탄환 소비

```text
TrySpendBullet(amount):
  1. 현재 탄환 >= amount인지 확인
  2. 부족하면 카드 사용 불가 처리
  3. 충분하면 현재 탄환 -= amount
  4. OnBulletSpent 이벤트 발행
  5. 현재 탄환 == 0이면 OnMagazineEmpty 이벤트 발행
```

### 탄환 획득

```text
GainBullet(amount):
  1. 현재 탄환 = min(현재 탄환 + amount, 최대 탄환)
  2. 실제로 증가한 수량만 반환
  3. OnBulletGained 이벤트 발행
```

### 탄창 비움 보상

탄환이 `1 이상 → 0`이 되는 순간, 턴당 1회만 발동한다.

> **빈 탄창의 반사신경**
> - 방어도 5 획득
> - 카드 1장 드로우
> - 시그니처 게이지 +1

의도:
- 탄창을 모두 쓰는 행동이 손해가 아니라 작은 성공 경험이 되도록 한다.
- 플레이어는 “마지막 탄환을 아끼기”보다 “다 쓰고 재장전하기”를 선택하게 된다.

### 턴당 발동 제한

```text
magazineEmptyTriggeredThisTurn = false
```

- 플레이어 턴 시작 시 `false`로 초기화한다.
- 탄창 비움 보상이 발생하면 `true`로 변경한다.
- 같은 턴에 재장전 후 다시 0발이 되어도 추가 발동하지 않는다.

# 4. 시그니처 게이지 규칙

## 4.1 기본 충전

마리아는 공통 규칙에 따라 카드 사용 시 시그니처 게이지를 +1 얻는다.

## 4.2 추가 충전 조건

| 트리거 | 게이지 변화 | 조건 |
|---|---:|---|
| 공격 카드가 피해를 1회 이상 적중 | +1 | 카드 1장당 최대 1회 |
| 탄창 비움 보상 발동 | +1 | 턴당 1회 |
| 재장전 카드 사용 | +1 | 재장전 카드 1장당 1회 |

### 중요: 다단히트 처리

다단히트 카드가 3회 적중해도, `공격 적중`에 의한 게이지 증가는 카드 1장당 최대 +1로 제한한다.

목적:
- 다단히트 카드가 시그니처 게이지를 과도하게 충전하는 문제 방지
- 카드 사용 + 적중 + 탄창 비움의 세 축이 모두 의미 있게 유지

## 4.3 시그니처 카드

게이지가 최대치(8)에 도달하면 아래 카드를 손패에 생성한다.

### 시그니처: `폭풍 연사`

| 항목 | 값 |
|---|---|
| 카드 ID | maria_signature_storm_burst |
| 종류 | Signature / Attack |
| 비용 | 0 |
| 대상 | 선택한 적 |
| 탄환 비용 | 0 |
| 보존 | 예 |
| 효과 | 선택한 적에게 4 피해를 6회 준다. 현재 탄환이 6발이면 각 타격 피해가 6으로 증가한다. 사용 후 탄환을 3발 획득한다. |
| 시그니처 게이지 | 생성 시 0으로 초기화 |
| 연출 | 화면 중심 조준선 → 빠른 총격 6회 → 마지막 피격 시 큰 누적 피해 숫자 |

### 구현 처리 순서

```text
1. 대상 적 유효성 확인
2. currentBullet == maxBullet 여부를 확인하여 hitDamage 결정
   - true: 6
   - false: 4
3. 대상에게 hitDamage 피해를 6회 적용
4. 각 타격 중 대상이 사망하면 남은 타격은 종료
5. GainBullet(3) 실행
6. 카드 소멸 처리 또는 버린 더미 처리
```

### 밸런스 의도

- 기본 피해: 4 × 6 = 24
- 탄창 최대 상태 보너스 피해: 6 × 6 = 36
- “재장전 직후 필살기 사용”이라는 간단한 최적화 포인트를 제공한다.
- 시그니처 자체는 탄환을 소모하지 않아, 탄환이 없어도 역전 카드 역할을 할 수 있다.

# 5. 상태값 및 이벤트 정의

## 5.1 MariaCombatState

```csharp
public class MariaCombatState
{
    public int CurrentBullet;               // 0~6
    public int MaxBullet = 6;

    public bool MagazineEmptyTriggeredThisTurn;
    public int AttackCardsHitThisTurn;      // 디버그/통계용
    public int ReloadCardsPlayedThisTurn;   // 디버그/통계용
}
```

## 5.2 권장 이벤트

```text
OnBulletSpent(int amount)
OnBulletGained(int amount)
OnMagazineEmpty()
OnReloadUsed()
OnMariaAttackCardHit(cardId, targetId)
OnMariaSignatureReady()
```

## 5.3 카드 사용 가능 조건

일반적인 마리아 공격 카드에는 `requiredBullet` 값을 둔다.

```text
CanPlayCard(card):
  1. 현재 에너지 >= card.energyCost 확인
  2. card.requiredBullet > 0이면 현재 탄환 >= requiredBullet 확인
  3. 대상 카드라면 유효한 대상 존재 여부 확인
  4. 모두 만족하면 true
```

UI 처리:

- 탄환이 부족한 카드는 회색 비활성화
- 카드 하단 또는 탄환 아이콘 옆에 `탄환 부족` 표시
- 카드 클릭 시에는 “탄환이 부족합니다” 토스트를 표시할 수 있음

  탄창 비움 보상 및 다단히트 시각 쾌감을 위해 사용하는 카드
- 향후 “다단히트 강화” 카드와 시너지를 만들 수 있음

| 장수 | 1 |

| 키워드 | 피니시 |

### 구현 순서

```text
1. currentBullet 값을 spentBullet에 저장
2. CurrentBullet = 0
3. damage = 6 + (spentBullet * 5)
4. 선택한 적에게 damage 적용
5. 탄창 비움 보상 조건 확인
   - 기존 탄환이 1 이상이었다면 OnMagazineEmpty 처리 가능
6. 공격 카드 적중 이벤트 발행
```

### 피해 예시

| 사용 전 탄환 | 피해 |
|---:|---:|
| 0 | 6 |
| 1 | 11 |
| 3 | 21 |
| 6 | 36 |

### 설계 의도

- “탄창을 비우는 행위”를 가장 강하게 보상하는 마리아의 대표 피니시 카드
- 탄환 0에서도 최소 피해 6은 보장한다
- 6발일 때 36 피해로 강한 폭발감을 제공한다
- 사용 직후 탄창 비움 보상(방어도 5, 드로우 1, 게이지 +1)을 받아 다음 행동으로 자연스럽게 이어진다

# 8. 카드 우선 처리 및 세부 규칙

## 8.1 공격 카드 적중 이벤트

마리아의 공격 카드가 피해를 1회 이상 성공적으로 적용하면 아래를 처리한다.

```text
OnMariaAttackCardHit(cardId, targetId):
  1. 해당 카드가 이번 사용에서 아직 게이지 적중 보상을 주지 않았는지 확인
  2. 시그니처 게이지 +1
  3. 해당 카드 사용의 적중 보상 처리 완료 플래그를 true로 설정
```

- 다단히트 카드는 피해가 여러 번 들어가도 게이지 +1만 획득한다.
- 방어도에 막혀 실제 체력 피해가 0이어도 “공격이 적에게 명중”했다면 적중으로 간주한다.
- 대상이 이미 사망해 공격 효과가 실행되지 않았다면 적중 이벤트는 발생하지 않는다.

## 8.2 탄창 비움 보상 발동 순서

탄환 소비형 카드가 탄환을 0으로 만들면:

```text
1. 탄환 소비 처리
2. 탄창 비움 여부 확인
3. 탄창 비움 보상 처리
   - 방어도 +5
   - 카드 1장 드로우
   - 시그니처 게이지 +1
4. 카드 본문 공격 효과 처리
```

이 순서를 권장한다.

이유:
- 마지막 탄환을 소모한 순간의 보상이 먼저 보이고,
- 이어지는 공격이 더 시원하게 느껴진다.
- 드로우된 카드가 현재 카드 처리 중 손패에 추가되어도 문제없다.

## 8.3 마무리 저격의 예외

`마무리 저격`은 탄환을 “전부 소비”한다.

- 현재 탄환이 0인 상태에서도 카드 사용 가능
- 탄환 0일 때는 탄창 비움 보상이 발동하지 않음
- 현재 탄환이 1 이상이면 탄창 비움 보상 발동 가능
- 탄창 비움 보상이 이번 턴에 이미 발동했다면 추가 발동하지 않음

# 11. UI/연출 요구사항

## 11.1 탄환

- 공격 카드 사용 전, 필요한 탄환 수만큼 탄환 아이콘이 짧게 강조된다.
- 소비 시 좌측부터 또는 우측부터 한 발씩 꺼지는 애니메이션을 재생한다.
- 탄환이 0이 되면:
  - 빈 탄창 효과음
  - 작은 화면 흔들림
  - `빈 탄창의 반사신경` 텍스트 팝업
  - 방어도/드로우/게이지 변화 UI 표시

## 11.2 공격 카드

| 카드 | 연출 키워드 |
|---|---|
| 정조준 사격 | 조준선, 단발 총성, 강한 단일 피격 |
| 속사 | 짧은 연속 총성, 2회 숫자 팝업 |
| 대충 쏘기 | 빠른 사격, 무작위 대상 이동 |
| 장전 사격 | 탄환 삽입 소리 후 단발 |
| 마무리 저격 | 탄환 수만큼 탄피 연출 → 큰 한 발 → 누적 피해 강조 |
| 폭풍 연사 | 화면 정지 0.2초 → 6연사 → 마지막 강한 반동 |

## 11.3 카드 비활성화

탄환 부족으로 사용할 수 없는 카드:

- 카드 전체 채도를 낮춘다.
- 비용 아이콘 옆에 빈 탄환 아이콘 또는 `탄환 부족` 라벨을 표시한다.
- 터치/클릭 시 선택되지 않도록 한다.
- 단, 마무리 저격은 탄환 0에서도 사용 가능하므로 비활성화하지 않는다.

# 12. 테스트 시나리오

## 12.1 기본 탄환 소비

1. 전투 시작: 탄환 6
2. 정조준 사격 사용
3. 예상 결과:
   - 에너지 3 → 2
   - 탄환 6 → 5
   - 적 체력 8 감소
   - 게이지: 카드 사용 +1, 공격 적중 +1

## 12.2 탄창 비움 보상

1. 현재 탄환 1, 손패에 정조준 사격 존재
2. 정조준 사격 사용
3. 예상 결과:
   - 탄환 1 → 0
   - 방어도 +5
   - 카드 1장 드로우
   - 게이지 +1 추가
   - 이후 정조준 사격 피해 8 적용

## 12.3 탄창 비움 보상 중복 방지

1. 현재 탄환 1, 탄창 비움 보상이 이번 턴 이미 발동함
2. 재장전으로 탄환 1 획득
3. 다시 정조준 사격 사용
4. 예상 결과:
   - 탄환은 0이 됨
   - 두 번째 탄창 비움 보상은 발동하지 않음

## 12.4 마무리 저격

1. 현재 탄환 6
2. 마무리 저격 사용
3. 예상 결과:
   - 에너지 2 소모
   - 탄환 6 → 0
   - 피해 36
   - 탄창 비움 보상 발동
   - 방어도 +5, 카드 1장 드로우, 게이지 +1

## 12.5 시그니처 카드

1. 게이지가 7인 상태에서 정조준 사격 사용
2. 카드 사용 +1로 게이지 8 도달
3. 예상 결과:
   - 손패에 폭풍 연사 생성
   - 게이지 0으로 초기화
4. 폭풍 연사 사용 시:
   - 탄환이 6발이면 6 피해 × 6회
   - 아니면 4 피해 × 6회
   - 사용 후 탄환 +3
   - 시그니처 카드는 전투에서 제거

# 13. 밸런스 확인 지표

마리아의 플레이 테스트에서 아래 항목을 기록한다.

| 지표 | 목적 |
|---|---|
| 턴당 평균 카드 사용 수 | 난사형 감각이 실제로 발생하는지 확인 |
| 턴당 평균 탄환 소비량 | 탄환 자원이 너무 빡빡하거나 느슨하지 않은지 확인 |
| 탄창 비움 보상 발동 횟수 | 고유 기믹이 충분히 노출되는지 확인 |
| 재장전 카드 사용 시점 | 플레이어가 자연스럽게 재장전을 이해하는지 확인 |
| 시그니처 평균 발동 턴 | 너무 늦거나 빠른지 확인 |
| 탄환 부족으로 카드가 막힌 횟수 | 답답함 발생 여부 확인 |
| 단일 적/다수 적 전투 승률 | 마리아의 역할 편차 확인 |

## 목표 감각

- 일반 전투에서 매 턴 최소 2~4장의 카드를 사용하는 빈도가 높아야 한다.
- 탄환 0 상태가 발생해도 1턴 이상 무력하게 멈추는 경우는 드물어야 한다.
- 탄창 비움 보상은 전투당 1~3회 정도 경험되는 것이 적절하다.
- 시그니처는 일반 전투에서 0~1회, 보스전에서 1~3회 사용하는 것을 초기 목표로 한다.
